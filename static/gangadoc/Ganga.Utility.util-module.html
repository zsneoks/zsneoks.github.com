execute_once()

source code  
Return True if this function was not yet executed from a certain line in the program code.
Example:
 execute_once() # -> True
 execute_once() # -> True

 for i in range(2):
  execute_once() # -> True (1st), False (2nd)

 if execute_once() and execute_once():  # -> False
